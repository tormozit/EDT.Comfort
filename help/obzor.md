# Обзор

**EDT Comfort (Комфорт)** — плагин для **1C:Enterprise Development Tools (EDT)** 2025.2+.

## Основные возможности

- **Навигация** — ссылки, переход к определению, последние места, наборы объектов, фильтр навигатора по набору.
- **Редактор BSL** — автодополнение, быстрая схема модуля, вставка со сравнением, команды ИР.
- **Отладка** — инспектор переменных, окно «Коллекция», пересчёт выражений, улучшенное отображение значений.
- **Сравнение конфигураций** — открытие объекта, навигатор, сравнение через ИР.
- **ИР** — автоподключение Инструментов разработчика Tormozit.
