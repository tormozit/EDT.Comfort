# Первые шаги

1. **Панель инструментов** — кнопки **Перейти**, **Ссылка**, **Последние**.
2. **Ctrl+F11** — скопировать ссылку на текущее место.
3. **Ctrl+3** — панель «Последние места».
4. **Параметры → Комфорт** — фильтры в списках, автодополнение, улучшения отладчика.
5. При работе с ИР — запустите отладку; см. [Интеграция с ИР](obshchie-mekhanizmy.md#integraciya-s-ir).
