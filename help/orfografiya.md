# Орфография

Проверка орфографии в редакторе кода и связанных полях на основе словарей **HUNSPELL** (русский и английский) ([#175](https://github.com/tormozit/EDT.Comfort/issues/175)).

## Требование JDT

Орфография Comfort использует **Eclipse JDT UI** (`org.eclipse.jdt.ui`). Сам плагин Comfort **ставится без JDT** ([#189](https://github.com/tormozit/EDT.Comfort/issues/189)); пока JDT нет — проверка орфографии отключена.

Если в EDT нет JDT: **Параметры → Комфорт** (или **Свойства проекта → Комфорт**) → ссылка **«Установить модуль орфографии (JDT)»** — откроется мастер установки с сайта Eclipse Releases. Отметьте **Eclipse Java Development Tools**, установите и **перезапустите EDT**.

## Подключение словарей

При **первом старте** (если JDT уже есть) плагин подключает поставляемые словари HUNSPELL для русского и английского языков и выставляет рекомендуемые настройки орфографии EDT. Штатная страница **Параметры → Общие → Орфография** переведена на русский.

Пользовательский словарь разделён на **локальный** (рабочая станция) и **проектный** (хранится с проектом). Добавление слова — через диалог **«Добавить в словарь»** или быстрые исправления.

![Страница «Орфография»: словарь Комфорт-HUNSPELL и флажок «Игнорировать сокращения в CamelCase»](images/issue-175-1.png)

## Разбор идентификаторов

В отличие от штатного токенизатора, слова со смешанным регистром разбиваются на подслова по границам **CamelCase** (заглавные буквы).

| Параметр | Где | Описание |
|----------|-----|----------|
| Проверять орфографию в идентификаторах в видимой области | **Параметры → Комфорт → Редактор кода** | Проверка имён в видимой части редактора |
| Игнорировать сокращения в CamelCase | **Параметры → Общие → Орфография** | Короткие фрагменты вроде аббревиатур в CamelCase не помечаются (включён по умолчанию) |

![Флажок «Проверять орфографию в идентификаторах в видимой области»](images/issue-175-2.png)

В панели **«Свойства»** проверка текста в поле ввода выполняется при получении полем фокуса.

На ошибочном слове доступны подсказки с вариантами замены, переключением между маркерами и пунктом **«Добавить в словарь»**.
