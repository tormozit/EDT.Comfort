Совместимость:
Проверено на EDT 2025.2

Установка:
Главное меню / Установить новое ПО... / Добавить / Архив... / Выбрать скаченный zip файл / Добавить / Пометить "Tormozit EDT" / Далее / Готово.
Если все сделали правильно, то EDT предложит перезапуск.

Обновление:
Если установщик показывает вам старую версию после выбора нового ахрива, то нужно нажать в нем "Управление..." и в открывшемся списке сайтов нажать "Обновить" или передобавить сайт с именем установочного файла.
Если все сделали правильно, то обновления плагина EDT предложит перезапуск.
