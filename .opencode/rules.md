# Project Rules

## 1. Язык
По умолчанию отвечать на русском языке.

## 2. Логи
Перед действиями в коде, который пишет в `debug-*.log` (проверить наличие `DebugCollectionAgentLog.log` в затронутых файлах), — прочитать последний `debug-*.log` из корня проекта.
Только после анализа лога задавать вопросы или делать правки.

## 3. Java
Java 17 обязателен. Путь:
```
C:\Program Files\1C\1CE\components\axiom-jdk-full-17.0.16+12-x86_64
```
Set JAVA_HOME to this path before any Java-related operations.

## 4. Maven
**ЗАПРЕЩЕНО запускать Maven.** Проект собирается через Tycho и требует полной Eclipse PDE target platform.
Если нужно проверить синтаксис — использовать `javac` с указанием JAVA_HOME, и только для одного файла.

## 5. Снапшоты перед правками
Перед каждым применением `edit` к файлу — создавать снапшот оригинального содержимого:
```
.opencode/snapshots/<YYYYMMDDHHMMSS>-<file-relative-path>
```
Команда копирования для PowerShell:
```
Copy-Item -LiteralPath "plugin/src/..." ".opencode/snapshots/<timestamp>-<basename>"
```

## 6. Откат через git
**ЗАПРЕЩЕНО** использовать `git checkout --` (revert) без явного разрешения пользователя.
Условия:
- Сначала проверить наличие снапшота в `.opencode/snapshots/`. Если снапшот есть — восстановить из него (копирование).
- Если снапшота нет — спросить пользователя, указав в вопросе слово **"гит"**.
- Пока пользователь не разрешит (не скажет "гит") — ничего не откатывать, не чистить, не удалять.
- При разрешении откатывать только те файлы, которые трогал в текущей сессии.
