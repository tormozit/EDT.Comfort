# Обзор

**EDT Comfort (Комфорт)** — плагин для **1C:Enterprise Development Tools (EDT)** 2025.2+.

## Основные возможности

- **Навигация** — ссылки, переход к определению, последние места, наборы объектов, фильтр навигатора по набору, команды из [Git-представлений](git-izmeneniya.md); «Перейти к определению» во все поля ввода со ссылками на метаданные ([#144](https://github.com/tormozit/EDT.Comfort/issues/144)).
- **Редактор BSL** — [автодополнение](avtodopolnenie.md) (в т.ч. подсказка по параметрам метода), быстрая схема модуля, команды ИР; [текстовые редакторы](redaktory-teksta.md) — быстрый поиск (Ctrl+F3), навигация, вставка со сравнением; серверные вызовы отдельным цветом ([#66](https://github.com/tormozit/EDT.Comfort/issues/66)); [проверка орфографии](orfografiya.md) словарями HUNSPELL.
- **Поиск** — [агрегация вхождений](rezultaty-poiska.md), колонка «Путь», флажок «Слово целиком» ([#102](https://github.com/tormozit/EDT.Comfort/issues/102)).
- **Отладка** — инспектор переменных, [Отладить объект ИР](obshchie-mekhanizmy.md#otladit-obekt-ir), «Выводить ИР» в параметрах точки останова ([#136](https://github.com/tormozit/EDT.Comfort/issues/136)), расширенное [окно «Коллекция»](okno-kollektsii.md), пересчёт выражений, улучшенное отображение значений.
- **Сравнение конфигураций** — открытие объекта, навигатор, **Сравнить в приложении ИР** (макеты табличного документа), двойной щелчок по модулям, кнопка **Развернуть**, неблокирующий поиск по дереву, кнопка **«Найти все»** ([#112](https://github.com/tormozit/EDT.Comfort/issues/112)), рекурсивное развёртывание ([#105](https://github.com/tormozit/EDT.Comfort/issues/105)).
- **ИР** — автоподключение Инструментов разработчика Tormozit; **Редактор ИР** для [табличных макетов](redaktor-tablichnogo-dokumenta.md), [СКД](skhema-komponovki-dannyh.md) и [редакторов запросов](redaktor-zaprosa.md) ([#150](https://github.com/tormozit/EDT.Comfort/issues/150)); «Найти ссылки ИР», «Проверить модуль ИР».
- **Фильтры** — [иерархический фильтр](obshchie-mekhanizmy.md#filtry-po-podstroke-v-spiskah) для списков, включая поле «Тип» ([#146](https://github.com/tormozit/EDT.Comfort/issues/146), [#147](https://github.com/tormozit/EDT.Comfort/issues/147)); флажок «Только помеченные» в окне редактирования типа ([#148](https://github.com/tormozit/EDT.Comfort/issues/148)).
