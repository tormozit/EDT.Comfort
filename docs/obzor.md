# Обзор

**EDT Comfort (Комфорт)** — плагин для **1C:Enterprise Development Tools (EDT)** 2025.2+.

## Основные возможности

- **Навигация** — ссылки, переход к определению, последние места, наборы объектов, фильтр навигатора по набору, команды из [Git-представлений](git-izmeneniya.md).
- **Редактор BSL** — [автодополнение](avtodopolnenie.md), быстрая схема модуля, команды ИР; [текстовые редакторы](redaktory-teksta.md) — быстрый поиск (Ctrl+F3), навигация, вставка со сравнением.
- **Поиск** — [агрегация вхождений](rezultaty-poiska.md) и колонка «Путь» в результатах глобального поиска.
- **Отладка** — инспектор переменных, [Отладить объект ИР](obshchie-mekhanizmy.md#otladit-obekt-ir), расширенное [окно «Коллекция»](okno-kollektsii.md), пересчёт выражений, улучшенное отображение значений.
- **Сравнение конфигураций** — открытие объекта, навигатор, **Сравнить в приложении ИР** (макеты табличного документа), двойной щелчок по модулям, кнопка **Развернуть**, неблокирующий поиск по дереву.
- **ИР** — автоподключение Инструментов разработчика Tormozit; **Редактор ИР** для [табличных макетов](redaktor-maketa.md) и [СКД](skhema-komponovki-dannyh.md).
