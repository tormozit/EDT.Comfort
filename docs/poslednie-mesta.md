# Последние места

Представление **Последние места** (категория Комфорт). Открывается по **Ctrl+3** или кнопке **Последние** на панели.

## Возможности

- Таблица недавно посещённых **методов** и **объектов метаданных** с фильтром по подстроке.
- Двойной щелчок или Enter — переход к месту.
- **Ctrl+T** — [показать объект в навигаторе](navigator.md).
- **Добавить в набор объектов** — перенести выделенные записи в активный набор.
- **История ИР** ([#113](https://github.com/tormozit/EDT.Comfort/issues/113)) — кнопка для просмотра истории коммита в Инструментах разработчика.
- Контекстное меню: копирование, удаление записей, переход.
