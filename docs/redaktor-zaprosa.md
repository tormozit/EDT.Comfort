# Редактор запроса

Под этим именем в документации объединены **три места** в EDT, где редактируется текст запроса:

| Место | Описание |
|-------|----------|
| **Редактор запроса** | окно "Редактор запроса"
| **Схема компоновки данных** | поле текста запроса в редакторе схемы компоновки
| **Динамический список** | поле текста запроса в окне "Динамический список"

Общие функции текстового поля (быстрый поиск, навигация по идентификатору, вставка со сравнением, контекстное меню) — [Текстовые редакторы](redaktory-teksta.md).

В модальном редакторе запроса Eclipse не доставляет стандартные привязки клавиш — Комфорт перехватывает **Ctrl+F3** / **Ctrl+Shift+F3** на уровне окна.

## Автодополнение

Во всех трёх редакторах запросов работает [автодополнение](avtodopolnenie.md) с умным фильтром и поддержкой ИР ([#132](https://github.com/tormozit/EDT.Comfort/issues/132)).

## ИР

При подключённом ИР доступны команды ([#150](https://github.com/tormozit/EDT.Comfort/issues/150)):

- **Форматировать текст ИР** для текста запроса.
- **Редактор ИР** — команда в контекстном меню, открывает редактор текста ИР в псевдомодальном режиме.

См. [Команды ИР](obshchie-mekhanizmy.md#komandy-ir-v-redaktore-bsl).
