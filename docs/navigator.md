# Навигатор

Дерево конфигурации и расширений.

## Возможности Комфорт

- [Фильтры по подстроке](obshchie-mekhanizmy.md#filtry-po-podstroke-v-spiskah) в поле поиска — здесь доступен и иерархический вариант (точка делит фильтр на секции «тип.имя»).
- **Фильтр по набору** — кнопка на панели навигатора (переключатель): показать только объекты **активного** набора; вытесняет штатный фильтр по подсистемам.
- **Показать в проводнике** — контекстное меню: проводник Windows на файле/папке элемента.
- **Комфорт → Добавить в набор объектов** — добавить выделенные объекты в активный набор.
- **Показать в структуре проекта** — контекстное меню: показать выделенный элемент в представлении **Структура проекта** ([#173](https://github.com/tormozit/EDT.Comfort/issues/173)).

## Двойной щелчок

- **Реквизит** (обычный, стандартный, общий, реквизит табличной части) — как в EDT: открывается редактор объекта МД с выделенным реквизитом; дополнительно активируется панель **Свойства** (она получает фокус последней) ([#140](https://github.com/tormozit/EDT.Comfort/issues/140), [#46](https://github.com/tormozit/EDT.Comfort/issues/46)).

## Команды только в этом окне

| Действие | Как вызвать |
|----------|-------------|
| Фильтр по набору | Кнопка «Фильтр по набору» на панели |
| Показать в проводнике | Контекстное меню |
| Добавить в набор | Контекстное меню → Комфорт |
| Показать в структуре проекта | Контекстное меню |

См. [Наборы объектов](nabory-obektov.md).
