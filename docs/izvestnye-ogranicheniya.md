# Известные ограничения

- Основная платформа — **Windows**.
- ИР требует установки в базе и корректной регистрации COM 1С.
- Часть команд ИР только при активном подключении приложения ИР.
- Переназначение Ctrl+F11/Ctrl+F12 вне контекста «В окнах» может конфликтовать с EDT — следуйте подсказкам в описаниях команд.

При проблемах: [Журнал Комфорт](zhurnal-komfort.md).
