# Установка и обновление

## Через интернет

1. **Справка** → **Установить новое ПО…** → **Добавить**
2. URL: `https://tormozit.github.io/EDT.Comfort/`
3. **EDT Comfort** → **Далее** → **Готово** → перезапуск EDT

## Без интернета

Скачайте ZIP с [релизов](https://github.com/tormozit/EDT.Comfort/releases) или используйте URL версии, например `https://tormozit.github.io/EDT.Comfort/1.0.0.13/`.

**Справка** → **Установить новое ПО…** → **Архив…** → выбрать ZIP.

## Обновление

Тот же URL. Если версия не видна: **Управление…** → **Обновить** у сайта или передобавить сайт.
