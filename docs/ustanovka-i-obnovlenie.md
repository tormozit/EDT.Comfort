# Установка и обновление

## Через интернет (рекомендуется)

1. Главное меню → **Справка** → **Установить новое ПО…** → **Добавить**.
2. Указать URL сайта обновления (последняя версия, composite):

   `https://tormozit.github.io/EDT.Comfort/`

3. Отметить **EDT Comfort** → **Далее** → **Готово**.
4. Если всё сделано правильно, EDT предложит **перезапуск** — согласитесь.

Корневой composite-URL `https://tormozit.github.io/EDT.Comfort/` содержит **все архивные линейки** — в «Детали установки» отдельные строки с номером версии (1.0.0.19, 1.0.0.16, 1.0.0.15, …). Фиксированная линейка — по URL вида `https://tormozit.github.io/EDT.Comfort/1.0.0.14/`. Полный перечень — на [главной странице сайта](https://tormozit.github.io/EDT.Comfort/).

## Без интернета (архив)

1. Скачать ZIP-файл с [релизов GitHub](https://github.com/tormozit/EDT.Comfort/releases) или с URL фиксированной версии, например:

   `https://tormozit.github.io/EDT.Comfort/1.0.0.13/`

2. Главное меню → **Справка** → **Установить новое ПО…** → **Добавить** → **Архив…** → выбрать скачанный ZIP-файл → **Добавить**.
3. Отметить **EDT Comfort** → **Далее** → **Готово** → перезапуск EDT.

## Обновление

Используйте тот же URL или архив новой версии.

Если установщик **не видит новую версию**:

1. В диалоге установки нажать **Управление…**
2. В списке сайтов нажать **Обновить** у сайта `https://tormozit.github.io/EDT.Comfort/`
3. Либо удалить сайт и **добавить заново** (при установке из архива — с именем установочного файла)

Плагин может показывать уведомление о доступной версии — см. [Настройки](nastroyki.md).
