# Справка EDT Comfort (Комфорт)

Плагин расширяет **1C:Enterprise Development Tools (EDT)** — навигацию, наборы объектов, редактор кода, отладку, сравнение конфигураций и доступ к **Инструментам разработчика Tormozit (ИР)**.

Команды плагина в подсказках помечены **(Комфорт)**.
