# Глоссарий

## ИР

**Инструменты разработчика Tormozit** — подсистема для платформы 1С. Комфорт подключает ИР из EDT через COM.

## EDT

**1C:Enterprise Development Tools** — среда разработки конфигураций.

## Комфорт

Плагин **EDT Comfort** (`tormozit.comfort`).

## Набор объектов

Именованный список объектов метаданных проекта для быстрого доступа и фильтрации навигатора.

## Навссылка

Текстовая ссылка на объект, модуль или строку. См. [Копирование ссылки](obshchie-mekhanizmy.md#kopirovanie-ssylki).

## p2

Формат сайта обновления Eclipse. URL: `https://tormozit.github.io/EDT.Comfort/`.

## Улучшать списки

Настройка Комфорт (**Параметры → Комфорт**): улучшенные фильтры по подстроке в списках EDT и доработки [панели «Результаты поиска»](rezultaty-poiska.md). Ранее называлась «Заменять фильтры по подстроке в списках».
