# Синхронизация базы

Диалоги синхронизации конфигурации с информационной базой при работе с ИР (динамическое обновление и др.).

## Уведомления Комфорт

При диалоге EDT **«Выбор действия»** (блокировка инфобазы для обновления):

- Если к базе **подключено приложение ИР** и в [приложениях отладки](prilozheniya-otladki.md) включена колонка **«Динамическое обновление»** — Комфорт автоматически выбирает **«Обновить динамически»** в диалоге блокировки.
- После **динамического обновления** — уведомление с предложением **переподключить приложение ИР** для обновления метаданных в сеансе ИР.
- При **исключительной блокировке** без возможности динамического обновления — предложение **отключить приложение ИР**, если оно блокирует обновление базы.

См. [Интеграция с ИР](obshchie-mekhanizmy.md#integraciya-s-ir), [Приложения отладки](prilozheniya-otladki.md).
