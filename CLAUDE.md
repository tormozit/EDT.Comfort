# Project rules

Все правила репозитория — в [AGENTS.md](AGENTS.md), единственном файле-источнике. Подключён импортом ниже — содержимое автоматически попадает в контекст при старте сессии.

@AGENTS.md

Проверка (в начале **следующего** чата, до любой правки): убедиться, что содержимое `AGENTS.md` реально развернулось в системном контексте (виден текст правил, а не просто строка `@AGENTS.md`).
