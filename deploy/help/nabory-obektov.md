# Наборы объектов

Представление **Наборы объектов** (категория Комфорт). Списки объектов метаданных, привязанные к **проекту** EDT.

## Назначение

- Хранить рабочие подмножества объектов конфигурации (задачи, рефакторинг, обзор).
- Фильтровать **навигатор** по активному набору.
- Быстро возвращаться к объектам из набора.

## Основные действия

- Создание и переименование наборов, переключение **активного** набора (отметка «●»).
- Таблица объектов: имя, путь; фильтр по подстроке.
- **Ctrl+T** — показать объект текущей строки в [навигаторе](navigator.md).
- Контекстное меню: копировать ссылку, удалить из набора.
- Добавление объектов из [навигатора](navigator.md) и [последних мест](poslednie-mesta.md).

## Фильтр навигатора

При включённом **Фильтре по набору** в навигаторе отображаются только объекты активного набора. См. [Навигатор](navigator.md).

## Открытие панели

Назначьте команду **Наборы объектов** в **Параметры → Клавиши** (категория Комфорт) или откройте через **Показать представление → Прочее → Наборы объектов**.
