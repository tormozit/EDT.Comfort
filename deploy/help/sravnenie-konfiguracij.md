# Сравнение конфигураций

Редактор сравнения EDT.

## Команды Комфорт

| Действие | Клавиши | Описание |
|----------|---------|----------|
| Открыть объект | F2 | Открыть выбранный объект в редакторе |
| Показать в навигаторе | Ctrl+T | Выделить объект в навигаторе |
| Сравнение через ИР | меню | При подключённом ИР |

Контекстное меню и подменю **Комфорт** — разворот дерева, дополнительные команды.
