# Отбор по подсистемам

Диалог **Отбор по подсистемам** (в коде EDT — «Фильтр по подсистемам») в навигаторе.

## Возможности Комфорт

- Автораскрытие ветки проекта по текущему выделению в навигаторе.
- Запоминание размеров и положения окна между сеансами.

## Наборы объектов

**В этом диалоге наборы объектов не показываются** — только штатное дерево подсистем. Фильтрация по набору — отдельная кнопка **Фильтр по набору** в [навигаторе](navigator.md); см. [Наборы объектов](nabory-obektov.md).

См. [Навигатор](navigator.md).
